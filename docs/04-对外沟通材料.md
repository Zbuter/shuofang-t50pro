## 项目简介

### 一句话介绍
> 硕方 T50 Pro 标签打印机控制程序是一款基于 Python 的开源蓝牙打印工具，让标签打印更简单、更高效。

### 完整介绍（100 字）
硕方 T50 Pro 控制程序是专为硕方 T50 Pro 标签打印机开发的第三方控制工具。采用蓝牙 4.0+ 连接，支持文字、二维码、条形码打印，提供标签模板设计器和批量打印功能。开源免费，支持 Linux/macOS/Windows 全平台，适用于资产管理、电缆标识、产品标签等多种场景。

### 详细版（300 字）
硕方 T50 Pro 标签打印机控制程序是一款开源的 Python 打印工具，旨在为用户提供更灵活、更高效的标签打印解决方案。

**核心功能**：
- 蓝牙无线连接，摆脱线缆束缚
- 支持文字、二维码、条形码多种打印类型
- 可视化标签模板设计器
- 批量打印支持 CSV 数据导入
- 跨平台支持（Linux/macOS/Windows）

**技术特点**：
- 基于 bleak 蓝牙库，稳定可靠
- SZPL 指令集，兼容硕方官方协议
- 模块化设计，易于扩展
- MIT 开源协议，免费商用

**适用场景**：
企业资产管理、IT 运维电缆标识、仓库货物标签、办公用品管理、产品包装标识等。

## 使用场景

### 场景 1: 企业资产管理

**需求**: 为办公设备打印资产标签，包含编号、名称、部门、二维码。

**方案**:
```python
template = LabelTemplate(width=40, height=30)
template.add_text("资产编号", font_size=16, bold=True)
template.add_text("ZC-2026-001", font_size=20)
template.add_text("使用部门：IT 部", font_size=14)
template.add_qrcode('{"id":"ZC-2026-001","name":"笔记本电脑"}')
```

**效果**: 批量打印 500+ 资产标签，效率提升 10 倍。

---

### 场景 2: IT 运维电缆标识

**需求**: 为机房电缆打印双向标识标签。

**方案**:
```python
template = LabelTemplate(width=36, height=24)
template.add_text("FROM: 配线架 A-12", font_size=14)
template.add_text("TO: 交换机 B-03", font_size=14)
template.add_barcode("CABLE-A12-B03", barcode_type="code128")
```

**效果**: 标准化电缆标识，故障排查时间减少 50%。

---

### 场景 3: 仓库货物管理

**需求**: 为入库货物打印含 SKU 和数量的标签。

**方案**:
```python
# 从 CSV 批量导入
# SKU,名称，数量，库位
# SP001,产品 A,100,A-01-02

template.add_text(f"SKU: {sku}", font_size=16)
template.add_text(f"名称：{name}", font_size=18, bold=True)
template.add_text(f"数量：{qty}", font_size=14)
template.add_text(f"库位：{location}", font_size=14)
template.add_qrcode(f"{sku}|{qty}|{location}")
```

**效果**: 入库效率提升 3 倍，出错率降至 0.1%。

---

### 场景 4: 实验室样品标识

**需求**: 为实验样品打印含日期、批次、二维码的标签。

**方案**:
```python
template.add_text("样品编号", font_size=14)
template.add_text(f"SAMPLE-2026-001", font_size=18, bold=True)
template.add_text(f"日期：2026-03-24", font_size=12)
template.add_text(f"批次：BATCH-03", font_size=12)
template.add_qrcode("SAMPLE-2026-001|2026-03-24|BATCH-03")
```

**效果**: 样品追溯完整，符合实验室管理规范。

## 常见问题应答

### Q1: 这是官方软件吗？

**答**: 不是。这是第三方开源工具，由社区开发者维护。与硕方官方 SDK 兼容，但不隶属于硕方公司。

---

### Q2: 使用这个工具会影响保修吗？

**答**: 不会。本工具仅通过标准蓝牙协议发送打印指令，不修改打印机固件或硬件，不影响官方保修。

---

### Q3: 支持哪些打印机型号？

**答**: 主要支持硕方 T50 Pro，其他 T50 系列理论上兼容。T60 及其他型号待测试。欢迎反馈测试结果。

---

### Q4: 可以打印图片吗？

**答**: 当前版本不支持直接图片打印。可通过二维码编码图片 URL 或数据。图片打印功能在规划中。

---

### Q5: 如何批量打印？

**答**: 使用 CSV 文件导入数据，配合批量打印脚本。示例：
```bash
python batch_print.py --input assets.csv --template asset_label
```

---

### Q6: 打印模糊怎么办？

**答**: 
1. 检查标签纸质量
2. 调整打印浓度（打印机设置）
3. 确保打印机头清洁
4. 尝试降低字体大小

---

### Q7: 连接不稳定怎么办？

**答**:
1. 确保打印机电池充足
2. 靠近打印机（5 米内）
3. 关闭其他蓝牙设备
4. 重启打印机蓝牙

---

### Q8: 可以用于商业用途吗？

**答**: 可以。本项目采用 MIT 开源协议，允许免费商用，无需支付授权费用。
