# 硕方 T50Pro 标签打印机技术调研报告

## 1. 设备概述

**硕方 T50Pro** 是硕方科技（北京）有限公司推出的便携式标签打印机，主要特点：

- **打印方式**：热转印/热敏
- **分辨率**：203 DPI（8 点/mm）
- **打印宽度**：最大 50mm
- **通信接口**：USB / 蓝牙 / WiFi（需确认具体型号配置）
- **适用场景**：线缆标签、设备标签、办公标签

## 2. 通信协议分析

### 2.1 接口类型

根据硕方同类产品技术特征，T50Pro 支持以下通信方式：

| 接口 | 协议 | 说明 |
|------|------|------|
| USB | CDC/Serial | 虚拟串口通信， baudrate 通常为 115200 |
| 蓝牙 | SPP/BLE | 蓝牙串口或 BLE GATT 特征值传输 |
| WiFi | TCP/IP | 网络 socket 连接（部分型号） |

### 2.2 打印指令集

硕方打印机通常采用 **TSPL**（Tag Printer Standard Language）或兼容指令集：

**基本指令格式**：
```
SIZE <width> mm,<height> mm
GAP <gap> mm
OFFSET <offset> mm
SPEED <speed>
DENSITY <density>
DIRECTION <0|1>
REFERENCE <x>,<y>
CLS
PRINT <count>,<copy>
```

**图形/文本指令**：
```
TEXT <x>,<y>,<font>,<rotation>,<x-mul>,<y-mul>,"<content>"
BARCODE <x>,<y>,<type>,<height>,<readable>,<rotation>,<content>
QRCODE <x>,<y>,<ecc>,<size>,<content>
BITMAP <x>,<y>,<width>,<height>,<data>
```

### 2.3 状态查询协议

打印机状态查询通常使用：
- `~T` - 获取打印机状态
- `~TS` - 获取传感器状态
- 返回值格式：ASCII 或十六进制

## 3. 技术选型

### 3.1 开发语言

**Python 3.8+** 作为主要开发语言：
- 跨平台支持（Windows/Linux/macOS）
- 丰富的串口/蓝牙库（pyserial、bleak）
- 易于封装和扩展

### 3.2 核心依赖库

| 功能 | 库 | 说明 |
|------|-----|------|
| USB 串口 | pyserial | 串口通信 |
| 蓝牙 | bleak | 跨平台 BLE 库 |
| 图像处理 | Pillow | 图片转位图 |
| 条形码 | python-barcode | 条形码生成 |
| 二维码 | qrcode | 二维码生成 |

### 3.3 架构模式

采用 **分层架构**：
```
┌─────────────────┐
│   应用层 API    │  ← 用户调用接口
├─────────────────┤
│   驱动层        │  ← 指令生成/解析
├─────────────────┤
│   通信层        │  ← USB/蓝牙/网络
├─────────────────┤
│   硬件层        │  ← 打印机设备
└─────────────────┘
```

## 4. 风险与挑战

### 4.1 已知风险

1. **协议不透明**：硕方官方未公开完整 TSPL 指令集文档
2. **固件差异**：不同批次固件可能存在指令兼容性差异
3. **蓝牙配对**：BLE 连接需要正确的 UUID 和服务发现

### 4.2 应对策略

1. **逆向分析**：通过官方软件抓包分析通信数据
2. **兼容模式**：实现指令集探测和自动适配
3. **单元测试**：建立模拟器进行无设备测试
