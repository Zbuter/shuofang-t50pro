## 2. 核心类设计

### 2.1 通信层 (Transport)

```python
# 抽象基类
class Transport(ABC):
    @abstractmethod
    def connect(self) -> bool: ...
    @abstractmethod
    def disconnect(self) -> None: ...
    @abstractmethod
    def write(self, data: bytes) -> int: ...
    @abstractmethod
    def read(self, size: int = 1024) -> bytes: ...
    @abstractmethod
    def is_connected(self) -> bool: ...

# USB 实现
class USBTransport(Transport):
    def __init__(self, port: str, baudrate: int = 115200): ...

# 蓝牙实现
class BLETransport(Transport):
    def __init__(self, address: str): ...

# TCP 实现
class TCPTransport(Transport):
    def __init__(self, host: str, port: int): ...
```

### 2.2 驱动层 (Driver)

```python
# TSPL 指令生成器
class TSPLCommand:
    def size(self, width: int, height: int) -> str: ...
    def gap(self, gap: int) -> str: ...
    def text(self, x: int, y: int, font: str, content: str) -> str: ...
    def barcode(self, x: int, y: int, type: str, content: str) -> str: ...
    def qrcode(self, x: int, y: int, content: str, size: int = 4) -> str: ...
    def bitmap(self, x: int, y: int, image: Image) -> str: ...
    def print(self, count: int = 1) -> str: ...
    def query_status(self) -> str: ...

# 图像转换器
class ImageConverter:
    @staticmethod
    def to_bitmap(image: Image) -> bytes: ...

# 状态解析器
class StatusParser:
```

### 2.4 主打印机类

```python
class Printer:
    def __init__(self, transport: Transport):
        self.transport = transport
        self.tspl = TSPLCommand()
        
    def connect(self) -> bool:
        return self.transport.connect()
    
    def disconnect(self):
        self.transport.disconnect()
    
    def print_job(self, job: PrintJob, count: int = 1):
        commands = job.render()
        commands += self.tspl.print(count)
        self.transport.write(commands.encode('gbk'))
    
    def get_status(self) -> PrinterStatus:
        self.transport.write(self.tspl.query_status().encode())
        response = self.transport.read()
        return StatusParser.parse(response)
```

## 3. API 设计

### 3.1 基础打印 API

```python
from printer_t50pro import Printer, USBTransport, PrintJob

# 创建打印机实例
transport = USBTransport('/dev/ttyUSB0')
printer = Printer(transport)

# 连接
printer.connect()

# 创建打印任务
job = PrintJob(width=50, height=30)  # 50mm x 30mm
job.add_text("设备标签", x=10, y=5, font="SIMSUN.TTF", size=24)
job.add_text("型号：T50Pro", x=10, y=15, font="SIMSUN.TTF", size=16)
job.add_qrcode("SN:20260324001", x=30, y=5, size=4)

# 打印
printer.print_job(job, count=1)

# 断开
printer.disconnect()
```

### 3.2 快捷打印 API

```python
# 文本打印
printer.print_text("Hello World", x=10, y=10, size=24)

# 条形码打印
printer.print_barcode("1234567890", type="CODE128", x=10, y=30)

# 二维码打印
printer.print_qrcode("https://example.com", x=10, y=50)

# 图片打印
from PIL import Image
img = Image.open("logo.png")
printer.print_image(img, x=10, y=70)
```

### 3.3 状态查询 API

```python
# 获取打印机状态
status = printer.get_status()
print(f"纸张状态：{status.paper_status}")
print(f"打印头温度：{status.head_temp}°C")
print(f"错误代码：{status.error_code}")
```

## 4. 数据流

### 4.1 打印流程

```
用户调用 → PrintJob 构建 → TSPL 指令生成 → 编码 (GBK) → Transport 发送 → 打印机执行
```

### 4.2 状态查询流程

```
用户调用 → TSPL 查询指令 → Transport 发送 → 打印机响应 → StatusParser 解析 → 返回状态对象
```
