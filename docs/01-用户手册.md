## 产品简介

硕方 T50 Pro 标签打印机控制程序是一个基于 Python 的蓝牙打印工具，支持：

- ✅ 文字打印（中英文）
- ✅ 二维码打印
- ✅ 条形码打印（多种制式）
- ✅ 自定义标签模板
- ✅ 批量打印

**核心特性**：
- 使用 SZPL 指令集（硕方自有协议）
- 蓝牙 4.0+ 连接
- 支持多种标签尺寸（12mm/15mm/18mm/24mm/36mm）

## 安装步骤

### 步骤 1: 克隆项目

```bash
git clone https://github.com/YOUR_USERNAME/shuofang-t50pro.git
cd shuofang-t50pro
```

### 步骤 2: 安装依赖

```bash
# 使用 pip 安装
pip install -r requirements.txt

# 或使用 pip3（某些系统）
pip3 install -r requirements.txt
```

### 步骤 3: 蓝牙配对

**Linux (使用 bluetoothctl)**:
```bash
# 启动蓝牙控制工具
bluetoothctl

# 在 bluetoothctl 中执行:
power on
agent on
default-agent
scan on
# 找到打印机后记录 MAC 地址
pair <MAC 地址>
trust <MAC 地址>
connect <MAC 地址>
quit
```

**macOS**:
1. 打开"系统偏好设置" → "蓝牙"
2. 开启打印机电源
3. 在列表中找到"T50 Pro"或类似名称
4. 点击"连接"完成配对

**Windows**:
1. 打开"设置" → "设备" → "蓝牙和其他设备"
2. 开启打印机电源
3. 点击"添加蓝牙或其他设备"
4. 选择打印机完成配对

### 步骤 4: 验证安装

```bash
# 运行扫描工具
python scan.py
```

如果能看到打印机列表，说明安装成功。

## 使用示例

### 基础打印

```python
import asyncio
from printer import T50ProPrinter

async def main():
    printer = T50ProPrinter()
    
    # 连接打印机
    await printer.connect("XX:XX:XX:XX:XX:XX")
    
    # 打印文字
    await printer.print_text("Hello T50 Pro!", font_size=24, bold=True)
    
    # 打印二维码
    await printer.print_qrcode("https://example.com")
    
    # 打印条形码
    await printer.print_barcode("1234567890", barcode_type="code128")
    
    # 断开连接
    await printer.disconnect()

asyncio.run(main())
```

### 打印中文标签

```python
await printer.print_text("产品名称：T50 Pro", font_size=20, align="center")
await printer.print_text("型号：TS-2024", font_size=16)
await printer.print_text("生产日期：2026-03-24", font_size=14)
```

### 打印资产标签

```python
# 打印资产编号
await printer.print_text("资产编号", font_size=18, bold=True)

# 打印二维码（包含资产信息）

await printer.print_qrcode(qr_data, size=4)

# 打印条形码

```

### 打印电缆标签

## 标签设计器

标签设计器提供高级模板功能，可组合多种元素。

### 创建模板

```python
from label_designer import LabelTemplate, LabelPrinter
import asyncio

async def create_label():
    # 创建 40mm x 30mm 标签模板
    template = LabelTemplate(width=40, height=30)
    
    # 添加文字
    template.add_text("产品名称", x=10, y=10, font_size=20, bold=True)
    template.add_text("型号：T50-Pro", x=10, y=35, font_size=16)
    
    # 添加二维码
    template.add_qrcode("https://example.com/product", x=10, y=55, size=3)
    
    # 添加条形码
    template.add_barcode("1234567890", x=10, y=85, barcode_type="code128")
    
    # 预览
    template.preview()
    
    # 打印
    printer = LabelPrinter()
    await printer.connect("XX:XX:XX:XX:XX:XX")
    await printer.print_template(template, copies=1)
    await printer.disconnect()

asyncio.run(create_label())
```

### 模板方法

| 方法 | 说明 | 参数 |
|------|------|------|
| `add_text()` | 添加文字 | text, x, y, font_size, bold, align |
| `add_qrcode()` | 添加二维码 | data, x, y, size |
| `add_barcode()` | 添加条形码 | data, x, y, barcode_type, height |
| `clear()` | 清空模板 | - |
| `preview()` | 预览内容 | - |

### 运行演示

```bash
python label_designer.py
```

## 故障排查

### 找不到打印机

**症状**: `scan.py` 显示"未找到打印机"

**解决方案**:
1. 确保打印机已开机（长按电源键 3 秒）
2. 确保打印机处于蓝牙配对模式（指示灯闪烁）
3. 在系统蓝牙设置中完成配对
4. 关闭其他可能占用打印机的应用
5. 重启打印机蓝牙（关机再开机）

### 连接失败

**症状**: 连接时超时或报错

**解决方案**:
```bash
# Linux: 检查蓝牙服务状态
systemctl status bluetooth

# 重启蓝牙服务
sudo systemctl restart bluetooth

# 查看已配对设备
bluetoothctl paired-devices

# 删除旧配对重新配对
bluetoothctl remove <MAC 地址>
```

### 打印没反应

**症状**: 连接成功但打印无输出

**解决方案**:
1. 运行调试工具探测正确的 UUID:
   ```bash
   python debug_bluetooth.py
   ```
2. 根据输出修改 `printer.py` 中的 `WRITE_UUID`
3. 检查标签纸是否安装正确
4. 确认指令格式（SZPL）正确

### 中文乱码

**症状**: 打印出的中文显示为乱码

**解决方案**:
1. 确保系统支持 GBK 编码
2. 检查 Python 文件编码声明：`# -*- coding: utf-8 -*-`
3. 在代码中明确指定编码：`text.encode('gbk')`

## 常见问题

### Q: 支持哪些操作系统？
**A**: 支持 Linux、macOS、Windows 10/11，需蓝牙 4.0+ 支持。

### Q: 可以打印图片吗？
**A**: 当前版本不支持直接图片打印，但可通过二维码/条形码编码数据。

### Q: 最大打印宽度是多少？
**A**: 支持 12mm/15mm/18mm/24mm/36mm 标签纸，默认配置为 40mm x 30mm。

### Q: 如何批量打印？
**A**: 使用 `print_template(template, copies=N)` 指定打印份数。

### Q: 蓝牙连接距离是多少？
**A**: 理论 10 米，建议 5 米内使用以保证稳定性。

### Q: 打印速度慢怎么办？
**A**: 
- 减少单次打印内容复杂度
- 确保打印机电池充足
- 靠近打印机减少蓝牙延迟
